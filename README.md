# Justin Coach — justinthepractice.com

Client website for Justin's coaching business. Pure static HTML/CSS/JS on GitHub Pages.

> ⚠️ **Live client site.** Pushing to `main` deploys immediately. Test before you push.

- **Live:** https://justinthepractice.com
- **Repo:** heathrowandrews/justin-coach
- **Status:** Production (auto-deploys via GitHub Pages, CNAME configured)

## Stack

- Pure static HTML/CSS/JavaScript — no build tools, no frameworks
- GitHub Pages hosting
- Google Fonts (DM Sans, Playfair Display)
- Design system: 8px spacing, purple accent `#7C3AED`

## Key files

| Path | Role |
|---|---|
| `index.html` | Homepage — hero, programs, resources, CTAs |
| `hyrox-quiz.html` | Interactive Hyrox Readiness Quiz (built by the Sprout bot) |
| `submit-testimonial.html` | Testimonial capture form |
| `programs/` | Program pages |
| `resources/` | Resource assets |
| `CNAME` | Custom domain config for GitHub Pages |

## Edit & deploy

No build step — edit the HTML/CSS/JS directly and open in a browser to preview. Push to `main` to deploy (GitHub Pages, live within ~1 min).
